# SCL Exploration Engine

## Présentation

Le SCL Exploration Engine est un moteur d'exploration interactif développé pour l'univers Stranger Code Legacy.

Son objectif est de permettre la création de scènes d'exploration immersives sans modifier le moteur lui-même.

Toutes les scènes sont décrites à l'aide de fichiers JSON.

---

## Fonctionnalités prévues

- Chargement dynamique des scènes
- Effet lampe torche
- Objets interactifs
- Inventaire
- Popups
- Effets sonores
- États des objets
- Transitions entre scènes
- Sauvegarde de progression

---

## Philosophie

Le moteur ne connaît jamais le scénario.

Il ne connaît que :

- une scène
- des objets
- des actions

Le contenu est entièrement décrit par les fichiers de données.
