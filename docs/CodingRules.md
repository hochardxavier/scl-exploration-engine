# Cycle de vie d'un composant

constructor()
→ Prépare les propriétés de l'objet.

init()
→ Initialise les ressources externes.

start()
→ Lance le fonctionnement.

update()
→ Met à jour la logique.

render()
→ Dessine.

destroy()
→ Libère les ressources (à venir).
