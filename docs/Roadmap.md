# ROADMAP

Le développement du SCL Exploration Engine suit une approche incrémentale.

Chaque fonctionnalité ajoutée doit être immédiatement exploitable dans une enquête.

---

# Court terme

## Exploration

- ✔ Chargement des scènes JSON
- ✔ Moteur de rendu
- ✔ Lampe torche
- ✔ Objets interactifs
- ✔ InputManager
- ✔ InteractionManager
- ✔ ActionManager
- ✔ PopupService
- ☐ InputPopupService

---

# Moyen terme

## Immersion

- Paramètres d'éclairage par scène
- Sons d'ambiance
- Effets sonores
- Animations
- Transitions
- Conditions d'interaction

---

# Long terme

## Outils de création

- Éditeur graphique de scènes
- Éditeur d'objets
- Prévisualisation en temps réel
- Générateur de missions
- Système de plugins
- Documentation du format de scène

---

# Vision

Le moteur doit permettre de créer une nouvelle enquête principalement en ajoutant :

- des images ;
- des fichiers JSON ;
- des ressources audio ;

sans modifier le moteur lui-même.
